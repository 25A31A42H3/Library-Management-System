📚 Library Management System in C
A simple Library Management System developed using the C programming language. This project helps manage library records such as adding books, displaying books, searching books, issuing books, and returning books.

✨ Features
➕ Add new books
📖 Display all books
🔍 Search books by ID
📕 Issue books
📗 Return books
💾 File handling for data storage
🖥️ Menu-driven interface
🛠️ Technologies Used
C Language
Structures
Functions
File Handling
Standard Input/Output

▶️ How to Run the Project
Step 1: Clone the Repository
git clone https://github.com/your-username/Library-Management-System.git
Step 2: Open Project Folder
cd Library-Management-System
Step 3: Compile the Program
Using GCC compiler:

gcc library.c -o library
Step 4: Run the Program
Linux / Mac
./library
Windows
library.exe
